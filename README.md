# espruino

Bluetooth HC-05 is linked over UART2:

| HC-05 | Espruino |
|-------|----------|
| VCC   | 3.3V     |
| GND   | GND      |
| TX    | RX (A3)  |
| RX    | TX (A2)  |
| Key   | B1       |

can choose UART1, but make sure the console thing is working correctly:
http://www.espruino.com/USART
